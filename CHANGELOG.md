# Changelog

## 1.5.1 - 2026-05-30

- Updated Chrome extension metadata for clearer screenshot annotation search positioning.
- Changed the default extension locale to English for better international store fallback.

## 1.5.0 - 2026-05-28

- Added arrow style variants: straight, curved multi-point arrows, and elbow arrows.
- Added arrow-style controls to the editor toolbar while the Arrow tool is active.
- Improved elbow arrows so multiple right-angle bends can be added before finalizing.
- Added axis snapping for elbow arrows to absorb small pointer drift during horizontal or vertical segments.
- Moved tooltips below icon buttons and documented double-click finalization for curved and elbow arrows.
- Replaced speech-bubble text background color with a fill color control for the bubble body.
- Hid the unused speech-bubble width/size control and auto-adjusted text color based on bubble fill brightness.
- Switched back to the Select tool after creating a speech bubble.
- Changed blur areas to default black filled masks with editable color.
- Doubled the default mouse-click stamp size and added localized left/right click labels below the cursor.
- Preserved arrow style metadata in canvas history and serialization.
